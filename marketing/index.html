Meridian

A scroll-driven memory timeline. Cards with photos appear as you scroll; a map in the background draws travel lines between them.

See the demo View source

What is this

Drop a folder of photos in, get a beautiful scrollable trip-map you can host anywhere.

Built originally as a personal birthday gift. Now open source.

How it works

  1. 1

    Drop in photos

    Point at a folder of trip photos. EXIF gets you dates and coordinates for free.

  2. 2

    Run meridian-triage

    The CLI extracts metadata, geocodes locations, and looks up historical weather. Outputs a Memory[] JSON.

  3. 3

    Host

    Point the app at the JSON, run bun run dev, and ship to any static host.
